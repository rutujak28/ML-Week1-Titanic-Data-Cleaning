# Titanic Survival Prediction - Data Cleaning Project

## Overview

This project focuses on data preprocessing using the Titanic dataset.

## Tasks Performed

* Loaded and explored the dataset using Pandas
* Identified missing values
* Filled missing Age values using Median Imputation
* Filled missing Embarked values using Mode Imputation
* Dropped Cabin column due to excessive missing data
* Encoded Sex using Label Encoding
* Encoded Embarked using One-Hot Encoding
* Visualized Age Distribution using Matplotlib
* Exported the cleaned dataset as a CSV file

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

## Output

* Cleaned Titanic Dataset
* Age Distribution Visualization
